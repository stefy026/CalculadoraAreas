Calculadora de Areas

Objetivos del Proyecto
•	Implementar una jerarquía de clases usando herencia
•	Aplicar polimorfismo para cálculos dinámicos
•	Crear una interfaz dinámica que se adapte a diferentes figuras
•	Soportar múltiples unidades de medida con conversión automática
•	Demostrar las ventajas de la POO en proyectos escalables
